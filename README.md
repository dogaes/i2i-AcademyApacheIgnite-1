# Apache Ignite 3

Single-node Apache Ignite 3 cluster accessed via the Java Thin Client, demonstrating IMDG + distributed SQL concepts.

## Projects

- Pull and run the Apache Ignite 3 container
- Expose the default thin client port (10800) to your local machine

- Create a Java project and include the Apache Ignite 3 Client dependency using the modern ThinClient API
- Create a Subscriber class for data model with customerId, dataUsage, smsUsage, callUsage
- Write a Java program that connects to local Ignite 3 node.
- Create a table named Subscriber
- Print the final state of all 5 subscribers to the console and terminate the program
  
## Stack

- Apache Ignite 3.1.0 (Docker)
- Java 17, Maven
- \`ignite-client\` Thin Client API
